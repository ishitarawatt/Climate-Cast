# Climate-Cast
ClimateCast is a modern, user-friendly weather forecasting website that allows users to check real-time weather conditions and a 5-day forecast for any city worldwide. It uses the OpenWeatherMap API and features a clean UI with smooth animations.

ClimateCast/
│
├── index.html        # Main HTML file
├── style.css         # Styling and UI design
├── script.js         # Weather logic & API calls
├── assets/           # Images, icons (optional)
└── README.md         # Project documentation
